## 项目介绍

这是我的个人品牌网站，目标是建立一个**工业生产级**的在线展示平台。

### 核心功能

- 展示个人资料和技能栈
- 分享项目案例和开发进度
- 自动同步 GitHub 仓库数据
- 完整的 SEO 和性能优化

### 技术亮点

使用 Next.js 15 App Router + TypeScript + Tailwind CSS + shadcn/ui 构建，部署在 Vercel，通过 GitHub Actions 实现持续集成。
