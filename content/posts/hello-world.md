## 为什么开始写博客

记录技术思考、项目复盘和学习笔记，是开发者成长最有效的方式之一。这个博客系统基于 **Next.js + Markdown + Decap CMS** 构建，所有文章都保存在 Git 仓库中，既方便版本管理，也能通过 CMS 后台在线编辑。

## 这里会写什么

- 前端工程化与 React 生态实践
- 数据可视化项目复盘
- 全栈开发中的踩坑记录
- 个人效率工具与学习方法论

## 技术栈

| 层 | 技术 |
|---|---|
| 框架 | Next.js 16 App Router |
| 样式 | Tailwind CSS 4 + 毛玻璃设计 |
| 内容 | Markdown + YAML frontmatter |
| 后台 | Decap CMS |
| 部署 | Vercel |

期待在这里持续输出有价值的内容。
